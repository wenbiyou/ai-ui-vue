# 组件文档

这里是 Ai UI Vue 所有组件的详细文档，每个组件都包含使用示例、API 说明和交互式演示。

## 核心组件

- [AiInput](./ai-input) - Prompt 输入框，专为 AI 对话场景设计
- [AiMessage](./ai-message) - 消息气泡，支持用户/AI/系统三种角色
- [AiLoader](./ai-loader) - 加载状态动画，三种样式可选
- [AiChat](./ai-chat) - 完整流式对话组件，开箱即用

## 功能组件

- [AiMarkdown](./ai-markdown) - Markdown 渲染，内置代码高亮
- [SmartForm](./smart-form) - AI 辅助智能表单，支持配置化字段和 AI 补全
- [StreamChart](./stream-chart) - 实时流式图表，基于 Canvas 动态数据推送
- [TokenCounter](./token-counter) - Token 计数与成本估算，支持主流大模型
